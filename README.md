# Autoencoder From Scratch

## 📌 Project Overview

This project implements an **Autoencoder neural network from scratch** using Python.
The goal is to learn compressed representations of data and reconstruct the original input.

The model is trained on the **MNIST dataset** to learn how to encode and decode handwritten digits.

## 🧠 How the Autoencoder Works

An Autoencoder consists of two parts:

1. **Encoder**

   * Compresses input data into a lower-dimensional representation.

2. **Decoder**

   * Reconstructs the original data from the compressed representation.

The model learns by minimizing **reconstruction loss**.
